# Getting Started

There is no official doc to get you started properly at the moment but we have a demo showcase at [https://github.com/mbret/prose-reader/tree/master/prose-react-native-demo](https://github.com/mbret/prose-reader/tree/master/prose-react-native-demo).

This project is made with expo and showcases downloading epubs, unzipping and serving them to a webview containing prose. It uses [prose-reader-react-native.md](prose-reader-react-native.md "mention") to bridge the app and prose.

Feel free to join the community if you are interested in building for React Native.
